How do I use useEffect for data fetching?

Here's how to properly fetch data with useEffect:

javascript 
useEffect(() => {
  const fetchData = async () => {
    const response = await fetch('/api/data');
    const data = await response.json();
    setData(data);
  };

  fetchData();
}, []);

Key points to remember:

  • • Always handle cleanup for subscriptions
  • • Consider using a library like React Query for complex cases
  • • Handle loading and error states

What about error handling?

You should wrap your fetch in a try-catch block to handle errors gracefully:

javascript 
useEffect(() => {
  const fetchData = async () => {
    try {
      setLoading(true);
      const response = await fetch('/api/data');
      if (!response.ok) {
        throw new Error('Network response was not ok');
      }
      const data = await response.json();
      setData(data);
    } catch (error) {
      setError(error.message);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);

What about cleanup functions?

Great question! Cleanup functions are essential for preventing memory leaks. You return a function from useEffect that React will call when the component unmounts or before re-running the effect
